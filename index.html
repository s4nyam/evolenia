Evo-Lenia Portal: Capturing Emerging Complexity in Lenia

What is Evo-Lenia portal?

Evo-Lenia is the supporting material for paper - "Capturing Emerging Complexity in Lenia" which is accepted in WIVACE2023

Here you will find all code and material to reproduce the paper work. It is divided into multiple parts. Each part corresponds to a Google Colab notebook where you can create your own version of code and run all cells along with other required offline material.
